Exporta el teu portafoli

Pots exportar el teu portafoli per mantenir els teus fitxers i contingut fora de línia. Pots crear una exportació en qualsevol moment i tantes vegades com vulguis. Quan exportes, rebràs un fitxer zip que conté el següent:

Lloc web HTML autònom: es creen fitxers que contenen un lloc web amb les dades del teu portafoli. No pots tornar a importar-ho a Mahara o a un altre sistema de portafoli, però es pot llegir en un navegador web estàndard i no necessites Mahara per veure'l. Per veure'l, extreu el fitxer zip i fes clic al fitxer index.html que hi ha a la carpeta HTML.

Leap2A: es crea un fitxer que conté una carpeta en el format estàndard Leap2A. A continuació, pots importar-ho a un altre sistema compatible amb Leap2A.
