Passos de Behat

Pots tenir una visió general ràpida dels passos de Behat disponibles i on s'utilitzen a les proves. Totes les proves es troben al directori /test/behat/features de la base de codi.

Les variables que es troben a les línies característica es representen amb "?" per agrupar aquestes línies més fàcilment. Les línies reals mostren les variables correctes.
