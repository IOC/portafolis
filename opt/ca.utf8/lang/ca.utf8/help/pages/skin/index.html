Skins

Els aspectes t’ajuden a personalitzar l'aspecte de les pàgines del teu portafoli per donar-los un toc personal.

Pots afegir fons, canviar els colors del text, els tipus de lletra, els botons i fins i tot, afegir el teu propi CSS personalitzat.

Si altres persones van crear aparences i les van posar a disposició per a tu, les veuràs aquí en aquesta pàgina. Pots preferir aparences d'altres per això pots utilitzar-les a les teves pròpies pàgines.

També pots importar aspectes des de altres instàncies de Mahara o exportar les teves aparences per compartir-les amb altres.
