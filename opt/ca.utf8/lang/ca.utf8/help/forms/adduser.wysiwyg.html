Editor HTML

Un editor HTML està disponible per a algunes seccions del lloc. Això es coneix com a editor "El que veus és el que obtens" (WYSIWYG). Això permet a la persona aplicar el format al text, p. negreta o cursiva. Si l'editor està desactivat, només poden introduir text sense format.

És possible que aquest paràmetre l'hagi desactivat un administrador del lloc.

Nota: els editors HTML poden aparèixer i funcionar de manera lleugerament diferent en alguns navegadors.
