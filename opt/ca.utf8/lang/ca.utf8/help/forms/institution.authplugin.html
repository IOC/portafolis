Connector d'autenticació

Aquí pots triar com s'autenticaran els membres d'aquesta institució al lloc. Les institucions tenen lligat almenys un mètode d'autenticació.

El connector d'autenticació interna comprova a la base de dades del lloc els noms d'usuari i les contrasenyes. Les persones han de tenir un compte perquè això funcioni, cosa que es pot fer permetent el registre mitjançant el formulari de registre o utilitzant Afegeix una persona o Afegeix persones per pàgines CSV.

El mètode d'autenticació cap permet entrar a qualsevol persona. La gent pot triar qualsevol nom d'usuari i contrasenya. Això només s'ha d'utilitzar per fer proves!

Els altres mètodes d'autenticació generalment s'autentiquen amb una font externa, com ara un servidor LDAP o una aplicació remota. Tenen la seva pròpia configuració, que pots configurar en una finestra emergent quan les afegeixis. Alguns mètodes estan desactivats perquè falten les extensions PHP necessàries o no es pot establir la connexió de sortida amb les fonts externes. Per exemple, el connector d'autenticació Persona necessita l'extensió PHP cURL i la connexió HTTPS a verifier.login.persona.org.

Una de les opcions que normalment comparteixen és si es poden crear comptes automàticament al lloc quan una persona inicia sessió per primera vegada (a diferència dels comptes creats prèviament per l'administrador, als quals la gent pot iniciar sessió).
