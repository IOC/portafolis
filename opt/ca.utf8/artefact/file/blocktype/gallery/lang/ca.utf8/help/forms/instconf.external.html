URL de la galeria externa

Per incrustar una galeria externa, has de copiar l'enllaç a la galeria externa i enganxar-lo al quadre de text.

És possible incloure les següents galeries externes:

  • Flickr, p. ex. https://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, p.ex. http://picasaweb.google.com/user.name/AlbumName#
  • Photobucket, p.ex. http://s1234.photobucket.com/albums/ee123/username/ o http://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Windows Live Photo Gallery, per exemple http://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

Nota:

  • Per utilitzar Flickr, l'administrador del lloc ha de proporcionar una clau API vàlida.
  • Per utilitzar Photobucket, l'administrador del lloc ha de proporcionar una clau API vàlida i una clau API secreta.
  • Perquè Windows Live Photo Gallery funcioni correctament, l'enllaç a una de les fotos dins de l'àlbum s'ha de copiar i enganxar al quadre de text per tal de mostrar tot l'àlbum.
