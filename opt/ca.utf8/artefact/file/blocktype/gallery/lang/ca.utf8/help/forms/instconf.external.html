Adreça URL de galeria externa

Per incrustar una galeria externa heu de copiar l'enllaç d'aquesta galeria externa i enganxar-lo en la casella.

És possible incrustar les següents galeries externes:

  • Flickr, p.ex. http://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, p.ex.http://picasaweb.google.com/user.name/AlbumName#
  • Panoramio, p.ex. http://www.panoramio.com/user/123456
  • Photobucket, p.ex. http://s1234.photobucket.com/albums/ee123/username/ or http://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Windows Live Photo Gallery, p.ex. http://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

Nota:

  • Per poder usar Flickr cal que l'administrador del lloc proporcioni una clau API vàlida.
  • Per poder usar Photobucket cal que l'administrador del lloc proporcioni una clau API i una clau API secreta vàlides.
  • Perquè Windows Live Photo Gallery funcioni correctament s'ha de copiar i enganxar l'enllaç a una de les fotos de l'àlbum en la casella corresponent perquè es mostri així l'àlbum sencer.
