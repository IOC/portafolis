Metadades SAML amb què es connecta el proveïdor d'identitat (IdP)

Aquest element requereix les metadades amb format XML per a l'IdP que vols per connectar-se.

Si afegeixes un IdP nou, selecciona "Afegeix un nou proveïdor d'identitat". "Proveïdors d'identitat disponibles" i afegeix les metadades rellevants al fitxer Camp "Metadades SAML del proveïdor d'identitat de la institució".

En cas contrari, selecciona un IdP existent a "Proveïdors d'identitat disponibles". Si fes qualsevol canvi a les metadades, s'actualitzarà per altres institucions que també utilitzen aquestes metadades. Per tant, vés amb compte amb el que fas canvi.
