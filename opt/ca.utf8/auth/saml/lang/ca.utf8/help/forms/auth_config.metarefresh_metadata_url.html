URL d'actualització de metadades

Si es proporciona un URL vàlid en aquest camp, el connector SAML baixarà automàticament les últimes metadades i les emmagatzemarà localment. Això elimina la necessitat d'actualitzar les metadades manualment quan canvien al vostre proveïdor d'identitat. Hauràs de tenir cron en execució per tal de comprovar les metadades.

Nota: les dades obtingudes d'aquest URL tindran prioritat sobre les metadades XML (si s'han introduït alguna) i les substituiran.
