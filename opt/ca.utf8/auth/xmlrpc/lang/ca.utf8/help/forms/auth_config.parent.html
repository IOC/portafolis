Autoritat parental

Si s'estableix una autoritat parental el usuaris tant podran entrar usant aquesta autoritat com via SSO.

Per exemple, podeu establir una autenticació LDAP que haurà de ser el "pare" d'aquesta autoritat. Això vol dir que els usuaris tant podran entrar via els formularis interns de Mahara, utilitzant els seus detalls LADP, com podran entrar via SSO.

No cal que establiu una autoritat parental. Si no ho feu els usuaris que utilitzin aquesta autoritat només podran accedir a a Mahara per SSO.
