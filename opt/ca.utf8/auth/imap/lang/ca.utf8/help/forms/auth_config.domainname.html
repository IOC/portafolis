Nom de domini

Accepta només adreces de correu electrònic d'aquest nom de domini. Aquesta opció és útil si utilitzes el teu propi nom de domini en un proveïdor IMAP compartit com Gmail o Outlook.com.

Per exemple, si has utilitzat el teu propi nom de domini a GMail, pots fer això:

Nom d'amfitrió o adreça: imap.gmail.com

Port: 993

Protocol: IMAP/SSL

Nom de domini: YOUR.DOMAIN
